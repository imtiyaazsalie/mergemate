# Data Privacy

## Your code stays yours

MergeMate never stores your source code. Never trains on it. Never shares it.

When you self-host, the only parties involved are you and your LLM provider. Pick the provider you trust, supply your own API key, and MergeMate stays out of the data path entirely.

---

## How the SaaS offering works

When using MergeMate's hosted service, code flows through our infrastructure strictly for processing — it's held in memory during the session and discarded immediately after. No logs, no storage, no persistence.

---

## What we collect

We track anonymous usage signals (command counts, feature adoption, error rates) to improve the product. This data contains **zero code content** and zero identifiable information.
