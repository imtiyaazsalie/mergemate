# Tools

MergeMate ships with a lean set of commands that plug directly into your PR workflow. Each tool targets a specific job — from generating descriptions to surfacing security gaps. Pick the ones that fit your team, automate the rest.

| Tool | What it does |
|---|---|
| **[`/review`](./review.md)** | Scans the diff and delivers structured feedback: bugs, security flags, review effort, and more. |
| **[`/describe`](./describe.md)** | Auto-generates PR titles, summaries, code walkthroughs, and labels. |
| **[`/improve`](./improve.md)** | Suggests concrete code changes — displayed as an interactive table or inline comments. |
| **[`/ask`](./ask.md)** | Answers free-form questions about the PR diff or specific lines. |
| **[`/add_docs`](./add_docs.md)** | Writes missing docstrings and comments for new or changed code. |
| **[`/generate_labels`](./generate_labels.md)** | Assigns labels to the PR based on the nature of the changes. |
| **[`/similar_issue`](./similar_issues.md)** | Finds related issues in the repository using vector search. |
| **[`/help`](./help.md)** | Lists every available tool with a checkbox for one-click triggering. |
| **[`/help_docs`](./help_docs.md)** | Answers questions by searching your repo's documentation folder. |
| **[`/update_changelog`](./update_changelog.md)** | Writes a changelog entry straight from the PR diff. |
